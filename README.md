# 🎮 Hangman Game — CodeAlpha Python Internship

A classic text-based Hangman game built with Python where the player guesses a hidden word one letter at a time.

## 📌 About the Project
This project is part of the CodeAlpha Python Programming Internship. The goal was to build a simple console-based Hangman game using core Python concepts.

## 🚀 Features
* Random word selection from a predefined word list.
* Tracks correct and incorrect guesses dynamically.
* Displays current word progress with blanks.
* Limits wrong guesses to 6 (classic Hangman rules).

## 🧠 Concepts Used
* `random` module (for picking the secret word)
* `while` loops (for the continuous game loop)
* `if-else` conditions (for checking guesses)
* Strings and Lists (for dynamic status tracking)
